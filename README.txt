1. Run the Makefile to compile the files.
2. To run the program, execute "./run.out" in the terminal after the files have been compiled.
3. Program detects if the tile coordinate entered are correct or not. 1 based indexing is used here.
4. Program detects whether the command is appended by a period or not.
5. Upon error, -1 is shown on stderr. Upon successful execution of a command, the board is shown in row major form along with tile names on stderr.
